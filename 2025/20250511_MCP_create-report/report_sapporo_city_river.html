1. 現地調査

1.1 豊平川

豊平川は札幌市を流れる主要河川であり、石狩川の支流です。市内の水源として重要な役割を果たしています。現地調査では、河川の状態、護岸整備状況、周辺環境などを確認しました。

調査日時： 2024年8月5日 13:15頃
調査区間： 豊平区平岸付近から中央区にかけての区間

写真1: 豊平川河岸の状況。護岸工事が適切に行われているが、一部に侵食の兆候が見られる。
写真2: 豊平川橋梁付近の流れの状況。水量は適切だが、上流からの土砂の堆積が見られる。
写真3: 河川敷の利用状況。市民の憩いの場として活用されているが、一部ゴミの散乱が見られる。

1.2 創成川

創成川は札幌市の中心部を流れる人工河川であり、都市景観の重要な要素となっています。現地調査では、水質、護岸整備、周辺の都市環境との調和などを確認しました。

調査日時： 2024年8月5日 14:30頃
調査区間： 北区北24条から中央区大通にかけての区間

写真4: 創成川公園付近の水路状況。水質は比較的良好だが、水面にごみが浮遊している箇所がある。
写真5: 創成川の護岸整備状況。都市型河川として適切に整備されているが、一部に老朽化が見られる。
写真6: 創成川と周辺建築物の調和。都市景観の一部として良好な状態を保っている。

1.3 新川

新川は石狩川水系の一級河川であり、札幌市の北西部を流れています。現地調査では、治水機能、生態系の状況、周辺農地との関係などを確認しました。

調査日時： 2024年8月6日 9:30頃
調査区間： 西区発寒から北区新琴似にかけての区間

写真7: 新川の河道状況。水量は適切だが、一部に土砂の堆積が著しい箇所がある。
写真8: 新川の護岸状況。コンクリート護岸と自然護岸が混在しており、一部に損傷が見られる。
写真9: 新川周辺の植生状況。外来種の繁茂が見られる箇所がある。

1.4 モエレ沼周辺水路

モエレ沼公園周辺の水路は、排水機能と景観機能を兼ね備えた重要な水系です。現地調査では、水路の連続性、水質、周辺環境との調和などを確認しました。

調査日時： 2024年8月6日 13:00頃
調査区間： 東区モエレ沼公園周辺

写真10: モエレ沼に流入する水路の状況。水質は良好だが、一部に水生植物の過剰繁茂が見られる。
写真11: 水路の護岸状況。人工的な整備がなされているが、環境との調和が図られている。
写真12: 水路と公園の接続部。排水機能と景観機能の両立が図られている。

2. 問題点の抽出

現地調査の結果から、以下の問題点が抽出されました。

2.1 豊平川

  1. 河岸侵食の懸念：一部区間で河岸侵食の初期兆候が見られ、将来的な護岸崩壊のリスクがある。
  2. 土砂堆積：橋梁付近で土砂の堆積が進んでおり、河道断面の減少による流下能力の低下が懸念される。
  3. ゴミの散乱：河川敷の一部でゴミの散乱が見られ、景観の悪化と生態系への悪影響が懸念される。

2.2 創成川

  1. 水質悪化：都市部を流れる特性上、水質の悪化が部分的に見られ、特に降雨後に顕著である。
  2. 護岸の老朽化：一部区間で護岸の老朽化が進行しており、長期的な安全性に懸念がある。
  3. 水面ゴミ：観光客や市民の多い地域であるにもかかわらず、水面にゴミが浮遊している区間がある。

2.3 新川

  1. 土砂堆積：数カ所で土砂の堆積が著しく、河道断面が減少している区間がある。
  2. 護岸損傷：特に自然護岸と人工護岸の接続部で損傷が見られ、洪水時の脆弱性が懸念される。
  3. 外来種の繁茂：河川敷に外来植物が繁茂しており、在来種の生育環境を圧迫している。

2.4 モエレ沼周辺水路

  1. 水生植物の過剰繁茂：水路の一部で水生植物が過剰に繁茂しており、排水機能の低下が懸念される。
  2. 接続部の設計：公園と水路の接続部において、安全対策が不十分な箇所が見られる。
  3. 冬季の機能：冬季の凍結時における排水機能の確保が課題である。

3. 提案事項

抽出された問題点に対して、以下の改善策を提案します。

3.1 豊平川

  1. 予防的河岸保護工の実施：侵食兆候が見られる区間について、予防的な河岸保護工（根固め工、護岸補強等）を実施する。
  2. 計画的な土砂浚渫：土砂堆積が見られる橋梁付近を中心に、計画的な浚渫を実施し河道断面を確保する。
  3. 環境美化活動の強化：地域住民や企業と連携した河川敷清掃活動を定期的に実施し、美化意識の啓発を図る。

3.2 創成川

  1. 水質モニタリングの強化：定期的な水質モニタリングを強化し、汚濁源の特定と対策を実施する。
  2. 護岸改修計画の策定：老朽化が進んでいる護岸について、中長期的な改修計画を策定し、計画的な更新を行う。
  3. ゴミ回収システムの導入：水面に浮遊するゴミを効率的に回収するシステム（例：スカムスキマー）の導入を検討する。

3.3 新川

  1. 河道内堆積土砂の定期的除去：土砂堆積箇所について、定期的な浚渫計画を策定し実施する。
  2. 護岸接続部の強化：自然護岸と人工護岸の接続部について、強化工事を実施し、洪水時の安全性を確保する。
  3. 外来種管理計画の策定：外来種の繁茂状況を定期的にモニタリングし、計画的な除去と在来種による植生回復を図る。

3.4 モエレ沼周辺水路

  1. 適切な水生植物管理：水生植物の繁茂状況を定期的にモニタリングし、過剰繁茂時には適切な除去を行う。
  2. 接続部の安全対策強化：公園と水路の接続部について、安全柵の設置や視認性の向上など、安全対策を強化する。
  3. 冬季対応設備の導入：冬季の凍結対策として、重要箇所に凍結防止装置の設置や融雪設備の導入を検討する。

以上の提案事項について、優先度と費用対効果を考慮した実施計画を別途策定することを推奨します。
